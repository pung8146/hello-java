객채(클래스) 는 속성과 기능을 가집니다.
자바에서는 속성을 필드(Field)라고 말하는 기능을 메소드(Method)라고 말합니다.

필드는 객체가 가지는 것

메소드는 객체가 가지는 기능을 말하는 합니다.
=================
public : 앞으로 배울 내용

void : 아무것도 나오는게 없다 

int : 정수

public void a() 
메소드 이름 : a
리턴 type : void
() : 파라미터가 없다

public void b(int 변수명)
메소드 이름 : b
리턴 type : void
(int 변수명 ) : 파라미터가 1개 있고  해당 파라미터의 type 은 정수(int)이다. 변수명은 그리 중요하지 않다

*메소드를 정의할 때 중요한것은 메소드이름 , 파라미터의 수, 파라미터의 type , 리턴 type

public int c() 
메소드 이름 : c
리턴 type : 정수
() : 파라미터가 없다
